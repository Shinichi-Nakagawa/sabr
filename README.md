====================
sabr
====================

SABRmetrics calculator for Python

Requirement
====================

python 2.7+

Install
====================

    $ pip install sabr

Usage
====================

```python
from sabr.stats import Stats
h9 = Stats.hr9(26, 209.7)  # Yu Darvish(2013) HR/9

```
